# 🏏 T20 Live Score Platform

Real-time T20 cricket live score website built using JavaScript and CricAPI.  
Responsive dark UI with PWA support and Google AdSense integration.

## 🚀 Features
- Live T20 match updates
- Auto refresh every 30 seconds
- Responsive glassmorphism UI
- PWA (Installable on mobile)
- Google AdSense ready

## 🛠 Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- CricAPI

## 📦 Deployment
Hosted via GitHub Pages.

## ⚠️ Note
API key is used for development/demo purposes.  
For production, move API key to backend.
